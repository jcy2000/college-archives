These tables show what values are in the database
Reviews
Name Definition
Text The actual text of the review that the user wrote.
Last Played Amount of time passed since user last played at time of review.
Time Played Last Two Weeks User's amount of time played in the last two weeks at time fo review.
Total Play Time User's amount of time played in total.
Comment Count Amount of comments on a single review.
Recommended Whether or not the review is rated as recommended.
Date Created Date the review was created, returned in local timezone as YYYY-MM-DD HH:MM:SS.
Date Updated Date the review was last updated. Returns date created if never updated.
Language What language the review was written in
Written During Early Access Whether or not the review was written during the game's early access time.
Play Time at Review User's total amount of time played at time of review.
Games
Name Definition
Game Name Official name of the game.
Owns
Name Definition
Game Attainment How the user attained the game.
Users
Name Definition
Reviews Made Amount of reviews the user has made.
Games Owned Amount of games owned by the user.
Votes
Name Definition
Amount Total amount of votes that a review has received.
Weighted Score A zero to one score based on votes for helpfulness. Higher score is better.
